GuacamoleClient

A lightweight desktop client for Apache Guacamole that runs remote sessions in their own app window and reduces keyboard shortcut conflicts between the remote session and your local computer.

Windows Installation

Stable channel for regular Windows installations.

WinForms Recommended

Best choice for most Windows users. Uses Microsoft WebView2, so browser security updates come through Windows/Edge.

Install WinForms Stable

Avalonia

Same app family as Linux and macOS. Uses bundled Chromium/CEF, so browser updates arrive with GuacamoleClient releases.

Download Avalonia for Windows

Linux Installation

Stable Debian/Ubuntu package for regular Linux installations.

Debian / Ubuntu

cd /tmp
wget "https://github.com/jochenwezel/GuacamoleClient/releases/download/__AVALONIA_DEB_RELEASE_TAG__/guacamoleclient-avalonia___AVALONIA_DEB_VERSION___amd64.deb"
sudo apt install ./guacamoleclient-avalonia___AVALONIA_DEB_VERSION___amd64.deb

Start it from the application menu or run guacamoleclient.

The package includes the .NET runtime and declares the required Linux desktop libraries as package dependencies.
